Quickstart

You need to install and configure a few tools to get started. Then download a theme. And carry on! Need detailed steps or videos?

Step 1: GitHub

All Volusion theme seed projects and files are available on GitHub. You will need an account and the client. Create a Git Hub account and install the client.

Step 2: Pick a Theme

You can download a Volusion Theme right through GitHub! We highly recommend using Method, a great looking eCommerce theme with faceted search and clean lines. Additional options are in the works! Just visit the Method theme page and either clone to your desktop or fork.

Step 3: Design Tools

Depending on your level of comfort and skill, you can install a minimum of tools to get started. Designers familiar with Bootstrap only need that knowledge. Of course, you can also install Bootstrap (v3.2), Ruby, and Sass and Compass.

Step 4: Development Tools

If you are developing API interactions and Apps, we recommend installing and configuring a few more tools including NodeJS and optional extensions like Postman REST Client and Web Developer.

Need Help?

Feel free to use this site for highlights and expanding information. We also provide links to our Apiary API reference, Stack Overflow, Uservoice, and Google Group at the top. You can also walk through all installation instructions and videos for these tools.

Can I use my own tools?

Absolutely!

If you have designer and developer resources that will work within this framework, you can use those tools to create , extend, and test. Make sure you always have the latest code through GitHub and ensure your preferred tools are compatible with the framework and file types supported by the Volusion theme seed projects and development.

We highly recommend using your tools with the listed tools in this guide. Volusion themes and interactions with the Theme Engine and Storefront API were developed within the Angular.js framework, with Node.js, Ruby, and all content saved to GitHub.
